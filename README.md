## 📦 我放了什么
- 个人简介、教育/实习经历、技能气泡  
- 作品集卡片（含文档 / 视频链接）  
- 图库与视频（支持 Lightbox 放大）  
- 游戏体验轮盘（可点击切换）  
- 联系方式与留言表单  

## 📁 资源放哪里
所有静态资源放在 public/ 下的对应目录：
- 图片：/images/
- 视频：/videos/
- 文档：/documents/

常用路径：
- 首页头像：/images/profile.jpg
- 作品 / 图库 / 游戏背景：按代码中的路径自行替换

## 🛠️ 主要技术
- React 18  
- Tailwind CSS  
- 自定义轮播、渐变、玻璃拟态等 UI 效果
